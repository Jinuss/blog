    # 图片裁剪插件cropper.js

    # 概述

    cropper.js是一个基于jQuery的图片裁剪插件，可以轻松实现图片的裁剪、缩放、旋转、移动等操作。 它提供了丰富的配置选项，可以满足各种不同的需求。 它的优点是简单易用，而且支持多种浏览器。
