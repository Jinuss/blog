`ES2020`的新特性如下：

- 动态导入 (Dynamic Import)
- `BigInt`大整数类型
- 空值合并运算符`??`
- 可选链操作符`?.`
- `Promise.allSettled`
- `String.matchAll`
- `globalThis`
- `import.meta`
- 模块命名空间导出
- `for-in`循环枚举标准化
